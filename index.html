Meeting Notes

React, Node, Rest API

This Reminder App is a ReactJS-based application designed to help users manage their notes and reminders efficiently. Built with Vite and React libraries, it provides a dynamic user experience with features like viewing, adding, editing, and toggling the completion status of action items in notes. The app leverages MaterialUI for a sleek, user-friendly interface and communicates with a Node.js backend via REST APIs for persistent data storage. Features: View Notes-Display all notes with titles and truncated content in a list view. Edit Notes-Enables editing of note content and action items directly within the expanded view. Toggle Action Items-Each action item comes with a checkbox to mark it as open or completed. Add New Notes-A create button allows users to add new notes with titles, contents, and action items. Automatic Timestamps-Newly created notes automatically include the date and time of creation. Search By Keyword-User can search through the title, content and action item by search

Basic Budget Calculator

React, Hooks

The React Budget Calculator is a web application designed to help users manage their monthly expenditures by categorizing them and calculating the necessary monthly income to ensure essential expenses, like rent, remain within a specified portion of their income. Developed for a programming challenge, the app allows users to input their monthly expenditures, specifying the name, amount, and category for each. The expenditures are categorized into Rent, Food, Utilities, Entertainment, and Miscellaneous, and displayed grouped by category. The application calculates the total amount spent in each category and determines the required monthly income to ensure that rent does not exceed 33% of the total income.

One Boston - Drug Regulatory

Java & Java Swing based Full Stack Project

The proposed approach involves developing a centralized platform that integrates bio-chemical compound data, disease information, and drug-related details. This platform should facilitate seamless communication between doctors, employees, and organizations, incorporating a role-based access system to ensure data security. Drug regulators can benefit from a dedicated module for monitoring and evaluating the safety and efficacy of drug compounds. Additionally, the system should include features for managing medical inventory within healthcare enterprises. A robust network infrastructure will underpin this solution, fostering collaboration, compliance, and information exchange, thereby advancing drug development and optimizing healthcare delivery in response to the evolving global health landscape.
